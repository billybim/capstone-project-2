Capstone Projek 2 - Membuat mini website portofolio

Tujuan:
Membuat websites untuk memperlihatkan skill, pengalaman dan ketertarikan. Berisi informasi tentang kamu dan backgroundmu.

Features:
- Hero section -> perkenalan dan background
- Resume Page -> education, work experience, skill dan informasi relevan lain (capstone projek 1)
- Section projek -> menampilkan projek yang telah dibuat
- About -> informasi tentang kamu dan ketertarikan kamu
- Contact -> informasi kontak kamu

Technical Requirement
- HTML
- CSS
- Website responsive
- images harus ada alt/ all text

Timeline:
- Jam pertama: mendapatkan konten dan ide desain. Buat kerangka dan mockup
- Jam kedua dan ketiga: buat struktur HTML dan desain css yg responsive
- Jam keempat: finalisasi desain dan konten, buat revisi dan optimaze performance
- Jam kelima: Launching web dan bagikan di Q&A.
